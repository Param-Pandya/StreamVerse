# Changelog

All notable changes to the StreamVerse streaming application will be documented in this file.

---

## [0.0.1] - 2026-07-16

### Added
*   **Centralized Profile Manager:** Stateful context hook supporting user active switching, kids mode flag checks, and inline renames.
*   **Cinematic Movie Details:** Backdrop gradients, horizontal chips, stateful add/remove from watchlist, and resume play parameters.
*   **Watchlist Persistence:** AsyncStorage-backed toggle handlers synchronizing results grid instantly.
*   **Continue Watching Scroller:** Simulated play triggers inside custom media player interface, progress bar metrics, and details page resume logic.
*   **Debounced Search Grid:** 300ms query keyboard input debounce, history badges, autocompletes, and paginated scroll loaders.
*   **Premium UX Polish:** Slide-up spring-animated Toast feedback, loading skeletons, and tactile AnimatedPressable scale feedbacks.

### Changed
*   **Branding Rewrite:** Refactored Disney+ references to the generic StreamVerse streaming portal identity.
*   **Expo 54 Migration:** Updated core assets, screen configurations, navigation stack libraries, and dependencies.

### Fixed
*   **Linter Checks:** Resolved Airbnb style exceptions, including nested ternary conditions, array index keys, and global method imports.
